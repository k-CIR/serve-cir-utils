# serve-mr-bids
This is a companion repository to: https://github.com/k-CIR/bids-utils-mr

If you clone **[bids-utils-mr](https://github.com/k-CIR/bids-utils-mr)** to your project folder on SPICE and *this* repository to your local machine you can run `ssh-connect.sh` to login to SPICE and automatically set up a simple interface to run [dcm2bids](https://unfmontreal.github.io/Dcm2Bids/latest/) on your data.

See the cir-wiki pages: https://k-cir.github.io/cir-wiki/mrc/mrc-bids/ for a detailed description on how to use the interface.
