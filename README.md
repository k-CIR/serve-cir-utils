# cir-utils-serve
This is a companion repository to: https://github.com/k-CIR/cir-utils

If you clone **[cir-utils](https://github.com/k-CIR/cir-utils)** to your project folder on SPICE and *this* repository to your local machine you can run `ssh-connect.sh` to login to SPICE and automatically set up a simple interface to structure and convert your data according to BIDS.

See the cir-wiki pages: https://k-cir.github.io/cir-wiki/SPICE/05_spiceBIDS/ for a detailed description on how to use the interface.
